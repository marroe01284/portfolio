SQUARE EYES movie streaming platform

The "SQUARE EYES" project is a movie streaming platform where users can explore, watch, and review a variety of films. Developed as part of a coding curriculum, it showcases various JavaScript tasks and assignments, providing an in-depth learning resource.

Project Breakdown

The "SQUARE EYES" project incorporates a range of functionalities designed to enhance user interaction and provide a seamless experience. The platform features a responsive design, ensuring optimal viewing across different devices. Dynamic movie cards allow users to interact with the content in real-time, while a cart functionality enables users to add and manage their selected movies easily. Additionally, a loader animation offers visual feedback during content loading, enhancing the overall user experience.
