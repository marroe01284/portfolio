The Blog Project

Welcome to The Blog project. This project contains the front-end user interface for an existing API blogging application. The application is designed to be responsive and dynamic, allowing users to view blog posts. This project includes admin pages for registering, logging in, and managing blog posts.

Project Breakdown

This project was about creating a responsive and user-friendly front-end for a blogging application. The main tasks included designing the user interface, integrating with an API, and implementing admin functionality for managing blog posts.
